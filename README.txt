# Brick Clash

## Description
*Brick Clash* est une version modernisée du jeu Crack-Attack pour le rendre actuel, tout en conservant un aspect visuel rétro, avec une palette de couleurs et des musiques rappelant les années de sa création en 1995. 

## Prérequis
- [CMake](https://cmake.org/download/) (version 3.10 ou supérieure)
- Un compilateur C++ (comme GCC, Clang, ou MSVC)
- [Git](https://git-scm.com/downloads) pour cloner le dépôt

## Cloner le Dépôt
```sh
git clone https://github.com/hamid-zen/crack-defense
cd crack-defense
```

## Générer les Fichiers de Build
1. Créez un répertoire build à la racine du projet :
```sh
mkdir build
cd build
```

2. Exécutez CMake pour configurer le projet :
```sh
cmake ..
```

## Compiler le Projet
```sh
cmake --build .
```

## Executer le jeu
```sh
./brick-clash
```
